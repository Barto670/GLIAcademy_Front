About Us

GLIAcademy is an educational platform that allows users to learn about the best globalisation, localisation and internationalisation practices

GLIAcademy also allows interaction between community members which can expand the knowledge about various topics

After creating an account the user is able to navigate through the platform and explore and learn in the following ways:

  • Theory based questions
  • Quizzes based on the theory learned
  • Simple programming tasks
